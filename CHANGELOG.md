# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/).

## v1.0.0 — Initial Release

- Dual timer modes: Normal (classic Pomodoro cycle) and Focused (stopwatch-style)
- Audio notifications: clock ticking, pre-end warnings, and completion chimes
- Rich analytics: day, week, and month recap views with visual timelines
- Customizable durations for work, short break, and long break
- JSON export/import for data backup and device migration
- Offline-first: all data stored locally via `chrome.storage.local`
- Privacy-focused: no data collection or external transmission
